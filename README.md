# Ejercicios Simples de Lógica y Algoritmia

Este repositorio compila una serie de scripts independientes enfocados en resolver problemas de lógica, manipulación de estructuras de datos y algoritmos fundamentales en Python. 

El objetivo de este espacio es mantener la práctica constante de la resolución de problemas y el pensamiento lógico, aplicando siempre buenas prácticas de legibilidad y eficiencia.

## 🛠️ Tecnologías
* **Python**: Uso de estructuras de control, funciones modulares y tipos de datos nativos.

## 📂 Contenido del Repositorio
Cada script resuelve un problema lógico aislado de forma autocontenida, ideal para repasar conceptos como:
* Estructuras de datos (listas, diccionarios, sets).
* Control de flujo y bucles óptimos.
* Modularización mediante funciones limpias.

---
*Lógica pura, código legible y resolución eficiente de problemas.*
